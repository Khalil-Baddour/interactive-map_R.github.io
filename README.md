# interactive-map_R.github.io
Création d'une carte interactive sur R : application sur la base de Prométhée des incendies de forêt en France 

Sur github sont téléchargeable seuls les fichiers compressés. Pour cela, des fichiers zip sont accopagnés. 
Ce code peut être copié et coller sur Rstudio, mais pour qu'il fonctionne il faut modifier le chemin du fichier vectoriel à importer et la légende selon vos propres données.
